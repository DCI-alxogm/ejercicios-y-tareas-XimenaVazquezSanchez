 Proceso para actualizar el repositorio en linea.
 primer paso. Crear una carpeta con el día y el mes en el que estamos. ejemplo: Jan29
 segundo paso. Crear dentro de esa carpeta un archivo. ejemplo: mi_primer_archivo.txt 
 tercer paso. Colocar el git add junto con el nombre del archivo: git add mi_primer_archivo.txt
 cuarto paso. Colocar el git commit y presionar enter (se abrirá una ventana)
 quinto paso. Al abrirse la ventana colocar un pequeño mensaje de actualización
 sexto paso. Dar enter, después ctrl+O para guardar, después enter, continuamos con ctrl+X para salir 
 septimo paso. Colocar el git push dar enter. Colocar tu usario dar enter, después pegar tu contraseña y dar enter.

Y listo por último sería entrar a tu repositorio y checar que todo este correcto con la actualización.
*/
